Bienvenidos a API Servicios

Acerca de Nosotros

API Servicios surge primero como un emprendimiento sencillo, enfocado en la ferreteria y venta de materiales para la contrucción en el año 2010, pero ante los conocimientos del fundador y algunos requerimientos de personas que requerian trabajos de plomeria y electricidad, ante esto se dio a la tarea de crear la empresa API Servicios que se enfocaba ya no solo en venta de productos ferreteros y materiales, si no tambien a hacer trabajos ante clientes ya sea particulares o empresas trabajos como, instalaciones electricas de cualquier indole ya sea en casas o en postes de luz asi como tambien intalaciones de plomeria desde 0 hasta reparaciones para cualquiera de las dos, ya sea electrica o plomeria.
